Com inicialitzar el projecte (previament node instalat):
1- Descarregar el repository
2- Accedir desde terminal a la carpeta de "frontend"
3- Executar la comanda "npm install"
4- Accedir a la carpeta "server2"
5- Executar la comanda "npm install"
6- Dins de la mateixa carpeta executar "npm start"
7- En un nou terminal anar a carpeta "frontend" i executar "npm start"
8- Accedir a "localhost:4200" desde cualsevol navegador
